ST-Ctrl v1.0.1 更新日志
最近更新：2026-06-09

━━━━━━━━━ 新增功能 ━━━━━━━━━

▶ 文件管理
完整的数据目录浏览器：支持多选、批量压缩/导出/删除、ZIP 压缩与解压。
内置文本编辑器，语法高亮 10+ 语言，带行号和撤销/重做。
大文件自动保护（超 512KB 禁止编辑）。

▶ 开机自启
手机重启后自动恢复酒馆服务（低功耗模式）。不需要手动打开 APP。

━━━━━━━━━ 功能改进 ━━━━━━━━━

▶ 备份与恢复
恢复失败时自动回滚，不会丢数据。备份/解压时跳过符号链接，
修复 zip-slip 路径穿越漏洞。恢复页面增加红色提示「恢复完成后需重启 APP」。

▶ Termux 数据迁移
修复 Android 11+ 下 Termux 备份文件在列表中不可见的问题。
脚本不再出现 CRLF 错误。一步生成脚本并复制命令，不再跳转设置页面。

▶ 世界书多选列表
全局世界书的选择框现在根据条目数量自适应高度，条目带圆角和选中高亮效果，
选择体验更接近浏览器。

▶ 自动备份
「保留几份」滑块现在可以精确选择 3 到 14 之间的每个整数，不再跳值。

━━━━━━━━━ 体验优化 ━━━━━━━━━

▶ 输入法
关闭输入法后不再出现黑色空白区域，页面恢复更丝滑。

▶ 控制台
修复滚动时「进入酒馆」卡片被顶栏遮挡的问题。卡片不再出现硬裁切。

▶ 权限
精简为一个「所有文件访问」权限，首次启动友好弹窗说明用途，
不再连续弹多个权限请求。

━━━━━━━━━ BUG 修复 ━━━━━━━━━

- 进入酒馆闪屏/首次向导被破坏：移除了会撕裂 ST DOM 的面板捕获系统
- 关闭后台酒馆后重进黑屏：状态复位修复
- 旋转屏幕后界面错乱：状态追踪修复
- 返回键失效：回调链修复
- node-bridge 线程挂起/数据竞争/假阳性崩溃：C++ 层稳定性修复
- SimpleDateFormat 多线程崩溃：升级为线程安全的 DateTimeFormatter
- 备份 renameTo 跨路径失败：增加磁盘空间检查和回退拷贝策略
- KeepAlive 闹钟 API 弃用：适配新版 Android
- WebView 性能模式失效：异步回调保护

━━━━━━━━━ 性能优化 ━━━━━━━━━

- V8 堆内存按模式限制（最高 256MB），减少 GC 卡顿
- CSS contain: layout style 缩小聊天消息重排范围
- WebView 空闲时取消 requestAnimationFrame，降低后台功耗
- 4 级性能模式控制 CPU 占用
- 构建：APK 体积缩小 35%，按 ABI 拆分打包

━━━━ v1.0.0 已知问题修复 ━━━━

- 选择背景时界面变白（ST 自身 bug，已在后续 ST 版本修复）
- WebView 缓存策略导致页面循环加载
- 恢复备份后首次启动闪退
- 文件选择器在模拟器上无法打开
- 启动时配置变更导致动画重播
